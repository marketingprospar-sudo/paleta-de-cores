# Método Recomeçar — Landing Page

Página de divulgação do evento **Método Recomeçar**, em Recife (24 de outubro), com a apresentação da paleta de cores da marca.

## Sobre

Landing page de página única, responsiva, construída em HTML e CSS puro (sem dependências de build). Inclui:

- Seção *hero* com logo animado, título e chamada para ação.
- Apresentação interativa da paleta de cores da marca, com nome, significado e código hexadecimal de cada cor.
- Seção final com nova chamada para ação e rodapé.
- Animações de entrada (fade-in via `IntersectionObserver`) e efeito de parallax suave.

## Paleta de cores

| Cor | Hex | Significado |
| --- | --- | --- |
| Marrom Profundo | `#200e03` | Acolhimento · Raiz · Profundidade |
| Verde Oliva | `#1d321c` | Renovação · Natureza · Crescimento |
| Bege Sofisticado | `#c3b09e` | Elegância · Suavidade · Luz |
| Branco Puro | `#ffffff` | Clareza · Recomeço · Espaço |

## Estrutura

- `index.html` — página completa (HTML, CSS e JS em arquivo único).
- `logo.png` / `logo_tree.png` — logotipos da marca.
- `.gitignore`

## Como visualizar

Abra o arquivo `index.html` em qualquer navegador, ou publique o repositório com o GitHub Pages.

Cormorant Garamond e Montserrat, via Google Fonts.
